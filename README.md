# Winzo-
This project is a frontend/backend prototype of a login system inspired by WinZO Games, designed to simulate user authentication, registration, and secure access to a gaming platform. 
The project includes features such as user signup, login authentication, password validation, and optional OTP-based verification. It is built for learning and demonstration purposes, focusing on clean UI/UX and scalable code structure.
